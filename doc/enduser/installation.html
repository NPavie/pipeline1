Daisy Pipeline Installation

This document describes how to get and install the Daisy Pipeline on your system

  • Prerequisites
  • Installation

Prerequisites

In order to run the Daisy Pipeline, you need to have the Java Runtime Environment (JRE) version 5 or later installed on your system.

The Java Runtime Environment can be downloaded from http://www.java.com.

Installation

  1. Download the zip release on the Pipeline GUI download page.

    Note: you have to select the zip archive corresponding to your environment.

  2. Create a directory on your local file system where you want to install the application.

  3. Extract the content of the zip archive to the newly created directory.

  4. Run the 'DAISY Pipeline' executable:

    • DAISY Pipeline.exe on Windows
    • DAISY Pipeline.app on Mac OS X
    • DAISY Pipeline on Linux

(Note that you probably want to create a shortcut to the Pipeline executable and put that on a convenient place (such as your desktop) for convenient access later on.)

Configuration

The DAISY Pipeline can be configured by manually editing the pipeline.properties located in the $INSTALL_DIR/plugins/org.daisy.pipeline_0.0.1/src/ directory (where $INSTALL_DIR is the application directory).

Amongst the interesting properties are:

dmfc.tempDir
The directory to store temporary files in.
dmfc.lame.path
The path to the LAME executable (Needed if you intend to run a script that includes MP3 encoding).
pipeline.python.path
The path to Python executable (Needed if you intend to run the RTF to DTBook script).
