Installing and configuring Lame on Windows

1. Download the Lame installer from one of the Lame executable download sites.

2. Extract the files of the archive you downloaded in step 1 to a directory of your choice, for example to C:\Program Files\lame\.

3.Locate the pipeline.properties file. This can either be accessed from your user interface, or via Explorer. If the latter, navigate to the Pipeline installation path (e.g. C:\Program Files\Daisy Pipeline\) and find pipeline.properties within that directory.

Open pipeline.properties in an XML or text editor, and make sure that the entry with the key dmfc.lame.path" points to the lame.exe file.

For example, if you installed Lame to C:\Program Files\lame\, the resulting element in pipeline.properties should read:


<entry key="dmfc.lame.path">C:\Program Files\lame\lame.exe</entry>

Save the pipeline properties file. All done.
