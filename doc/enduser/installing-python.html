Installing and configuring Python on Windows

1. Download the Python installer for Windows from the download section of the Python website.

2. Install the Python runtime by double clicking on the installer (.msi) file that you downloaded in step 1.

3. When the installer asks you for a destination directory, make a note of it for use in step 4.

When the installer has finished, Python is installed on your system, and all that remains is to make the pipeline aware of the installation path as selected and noted during step three.

4.Locate the pipeline.properties file. This can either be accessed from your user interface, or via Explorer. If the latter, navigate to the Pipeline installation path (e.g. C:\Program Files\Daisy Pipeline\) and find pipeline.properties within that directory.

Open pipeline.properties in an XML or text editor, and make sure that the entry with the key "pipeline.python.path" points to the python.exe file within the python installation directory.

For example, if you installed Python to C:\Python25\, the resulting element in pipeline.properties should read:


<entry key="pipeline.python.path">C:\Python25\python.exe</entry>

Save the pipeline properties file. All done.

Installing and configuring Python on Linux

Python is most likely already installed on your system. If not, you need to install it manually. This process is out of scope for thsi document since the method differs for each Linux distribution, but Ubuntu users will probably use the Synaptic package manager, Fedora users can simply type yum install python with root privileges while Debian users need to type apt-get install python. Refer to the documentation of your Linux distribution for details.

To find the path to your python interpreter, type which python in a terminal window. If you run the GUI version of the Daisy Pipeline, you need to add this path to the 'Paths' page in the preferences. If you run the command line version of the Daisy Pipeline you need to edit the pipeline.user.properties file in the root folder of the Daisy Pipeline installation and edit the pipeline.python.path property.

For example, if Python is installed in /usr/bin/python, the resulting element in pipeline.properties should read:


<entry key="pipeline.python.path">/usr/bin/python</entry>
