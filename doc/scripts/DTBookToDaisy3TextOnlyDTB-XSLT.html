Overview

Converts a valid DTBook 2005 document to a text-only DAISY 3 book.

  • The generated DAISY book will have a spine composed of only one big SMIL document.
  • The conversion does not support DTBook documents with embedded MathML.

Configuration

Input DTBook
Required. The input DTBoook document; the grammar version needs to be in the 2005 series.
Output OPF
Required. The path to the OPF manifest of the resulting DAISY 3 book.
