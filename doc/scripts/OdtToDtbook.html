Overview

This script converts an ODT file to a DAISY 3.0 XML, and then validates the result.

Input Requirements

An ODT file. To ensure correctness of the DAISY 3.0 XML output, make sure to follow the ODT authoring guidelines.

Output

A DAISY 3.0 XML file, which may or may not be valid - be sure to check the result of the validation pass!

Configuration

Required. Input ODT file
Required. Output DAISY 3.0 XML file
Optional. XML Validation Report
Select a file to store the DTBook validation XML report in. Note that information on the validation result is always shown in the user interface; regardless of whether you activate this option or not.
