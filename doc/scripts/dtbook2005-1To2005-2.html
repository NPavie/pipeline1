Overview

This script takes a DTBook document valid to version 2005-1 of the DTD and creates a new version that is valid to the 2005-2 DTD.

Input Requirements

This script requires a valid DTBook 2005-1 document as input.

Output

The output of this script is a copy of the input document, now valid to DTBook 2005-2. Note that this transformer will remove all style attributes from the input document. No other destructive action is taken. It may rearrange the order of doctitle, covertitle, and docauthor elements within frontmatter. It will not add any new information to the input document.

Configuration

Input DTBook 2005-1
Required. The input DTBook 2005-1 document.
Output DTBook 2005-2
Required. The output DTBook 2005-2 document.
